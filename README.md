# Movie_Genres_CNN_Classification
Este projeto tem como objetivo classificar uma base de dados de imagens de posters de filmes, rotulados como Ação, Comédia e Drama, com o objetivo classificar os porters pelo seu respectivo genero. Os dados foram obtidos via API do site TMDB. O código para extrair os dados pode ser encontrado neste outro projeto [link](https://github.com/Luizgs7/Movies_DataSet)
